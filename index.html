☁️ Connect Firebase
Permanent cloud storage — data never lost, syncs across all devices.
⚡ STEP 0 — HOST THE APP ONLINE FIRST Firebase needs the app to be on the internet (not local file):
1. Go to app.netlify.com/drop on your laptop
2. Drag and drop the deporrh.html file onto the page
3. You get a free link like random-name.netlify.app
4. Open THAT link on your phone — then connect Firebase below
5. Add it to Home Screen from the Netlify link
HOW TO SET UP: 1. Go to console.firebase.google.com
2. Create project → name it "deporrh"
3. Realtime Database → Create → Test mode
4. Settings ⚙️ → Your apps → Web → Register app
5. Copy the config values below
